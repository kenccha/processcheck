1.2 역할 소개

ProcessCheck에는 3가지 역할이 있습니다:

  • 실무자 (Worker) — 체크리스트 작업 수행, 완료 보고
  • 매니저 (Manager) — 부서별 작업 검토, 승인/반려
  • 기획조정실 (Observer) — 전체 현황 모니터링, 게이트 승인

1.3 대시보드 둘러보기

로그인 후 첫 화면은 대시보드입니다. 통계 카드, 작업 목록, 알림을 한눈에 확인할 수 있습니다.
